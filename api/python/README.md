# Airthings API SDK for Python

API SDK for Airthings consumer products.

Prerequisites:

- [Python](https://www.python.org/downloads/) 3.10+
- [Poetry](https://python-poetry.org/docs/#installation)

Install dependencies:

```bash
poetry install
```
